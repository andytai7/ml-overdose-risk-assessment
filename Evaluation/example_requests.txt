## Test with Missing Features (Will Return Error)
curl -X POST http://localhost:8000/predict/overdose \
  -H "Content-Type: application/json" \
  -d '{
    "ost": "value",
    "Opioiduse": "value"
  }'

# This will return:
# {
#   "error": "Missing required features",
#   "required_features": 48,
#   "provided_features": 2,
#   "missing_features": ["rheumatoid_arthritis_date_b", "parkinsonism_date_b", ...]
# }
